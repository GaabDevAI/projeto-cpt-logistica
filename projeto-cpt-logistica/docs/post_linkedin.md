# Sugestao de post para LinkedIn

Desenvolvi um dashboard logistico em Python para analisar uma agenda de viagens
por turno, destino e veiculo.

O projeto simula um fluxo real de operacao: leitura de CSV, tratamento de
horarios, classificacao em T1/T2/T3, consolidacao de origem e destino,
indicadores e visualizacoes interativas.

Para preservar privacidade e seguir boas praticas de dados, toda a base usada
no repositorio e ficticia. A estrutura representa um cenario operacional, mas
nao contem informacoes reais de pessoas, veiculos, rotas ou empresas.

Principais pontos tecnicos:

- Python
- Pandas
- Streamlit
- Plotly
- Tratamento de datas e turnos
- Dados sinteticos para portfolio
- Exportacao de CSV filtrado
- Geracao opcional de planilha Excel

Esse projeto reforcou como um bom tratamento de dados pode transformar uma
exportacao operacional simples em uma visao clara para tomada de decisao.
